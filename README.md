# Spotify Clone

A frontend clone of Spotify built using **HTML**, **CSS**, and **Vanilla JavaScript**. This project recreates Spotify's desktop interface with dynamic content rendering, interactive UI elements, and smooth animations.

## Features

* Spotify-inspired desktop interface
* Dynamic rendering of albums, artists, trending songs, and recently played sections
* Interactive footer player
* Sticky filter bar with scroll effects
* Hover animations and transitions
* Reusable JavaScript rendering function

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap Icons

## What I Learned

* Building complex layouts with Flexbox
* Creating reusable JavaScript functions
* Dynamically rendering UI using arrays and `forEach()`
* DOM manipulation and event handling
* Managing UI state with JavaScript
* CSS positioning, transitions, and hover effects
* Organizing project files for better maintainability
